Javascript generated Navigation Menu using Template Literals

Rather unorthodox and most likely breaking a lot of rules and good practices of proper web development solution for implementing a dynamically generated navigation menu in an otherwise static multipage HTML site.

See JS file functions.js for how it works. That is where I build the menu.

I simply was looking for some solution -- any solution -- to create an easy to control and update navigation for my Github pages.

Likely doesn't work across all browsers, but certainly does in the ones I care for. Fallback for non-JavasScript situations is simple: the 'Home' menu item is hardcoded in the HTML and the index.html should show all pages of the site in the content area to navigate to, when teh menu is not there.

It is not elegant, but workable, and only intended for my own use (but you may use it if you like).
